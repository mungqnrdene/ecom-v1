🕒 5-Minute Refund Window

Implementation Overview & Testing Guide

✨ Feature Overview

Захиалга үүссэнээс хойш зөвхөн 5 минутын дотор буцаалт хийх боломжтой.

5 минут дотор
  • Буцаах товч идэвхтэй
  • Цагийн тоолуур харагдана
  • Буцаалт амжилттай
5 минут өнгөрсөн
  • Товч идэвхгүй болно
  • "Хугацаа дууссан" гарна
  • Backend буцаалт зөвшөөрөхгүй

🔧 Backend Validation

OrderController.php - refund() method:

// Calculate deadline $refundDeadline = $order->created_at->copy()->addMinutes(5); // Check if expired if (now()->greaterThan($refundDeadline)) { return back()->withErrors([ 'error' => 'Буцаалтын хугацаа дууссан байна...' ]); } // Process refund $order->update([ 'status' => 'refunded', 'payment_status' => 'refunded', 'refunded_at' => now(), ]);

🎨 UI Logic (Blade Template)

Conditional button display:

@php $refundDeadline = $order->created_at->copy()->addMinutes(5); $canRefund = now()->lessThanOrEqualTo($refundDeadline) && $order->status !== 'refunded'; @endphp @if($canRefund) <!-- Show active refund button --> <button type="submit" class="btn btn-danger"> Буцаах </button> @else <!-- Show disabled button --> <button class="btn btn-secondary" disabled> Хугацаа дууссан </button> @endif

⏲️ Live Countdown Timer

JavaScript countdown on order details page:

// Initialize timer let timeLeft = {{ $timeRemaining }}; const timer = setInterval(() => { timeLeft--; if (timeLeft <= 0) { clearInterval(timer); location.reload(); // Refresh to show expired } else { // Update countdown display (MM:SS) const minutes = Math.floor(timeLeft / 60); const seconds = timeLeft % 60; countdownEl.textContent = `${minutes}:${seconds}`; } }, 1000);

💾 Database Testing

To test expired orders, manually update created_at:

-- Make order appear 10 minutes old UPDATE orders SET created_at = NOW() - INTERVAL 10 MINUTE WHERE id = 1; -- Or use Tinker php artisan tinker $order = Order::find(1); $order->created_at = now()->subMinutes(10); $order->save();
